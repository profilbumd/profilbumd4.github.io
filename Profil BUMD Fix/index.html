Aplikasi Profil BUMD

Aplikasi Profil BUMD merupakan sebuah aplikasi sederhana berbasis website untuk memudahkan entitas pemeriksaan / pemerintah daerah dalam melakukan input data terkait BUMD di masing-masing daerah. Aplikasi ini diharapkan dapat meningkatkan efektivitas dan efisiensi pemenuhan permintaan data profil BUMD di lingkup kerja BPK Perwakilan Provinsi Jawa Tengah.
